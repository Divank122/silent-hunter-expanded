# Ultimate Silent Card Expansion

为《杀戮尖塔2》静默猎手添加新卡牌的模组。

## 作者

- **卡牌设计 & 美术设计**：原始小金人
- **编码实现**：Divank

## AI使用声明

本项目在开发过程中使用了AI辅助工具进行代码编写、调试与美术绘制，所有AI生成内容均经过人工审核与调整。

## 依赖

- [BaseLib](https://github.com/Alchyr/BaseLib-StS2) 3.0.6+

## 安装

将以下文件放入游戏的 `mods/UltimateSilentCardExpansion` 文件夹中：
- `UltimateSilentCardExpansion.dll`
- `UltimateSilentCardExpansion.pck`
- `UltimateSilentCardExpansion.json`

## 卡牌列表

### 罕见卡牌

#### 疾风步 (Gust Step)
- **类型**: 技能
- **耗能**: 1
- **描述**: 丢弃所有手牌。抽3张牌。
- **升级**: 抽牌数提升至4张。

#### 蛛网 (Spider Web)
- **类型**: 技能
- **耗能**: 1
- **描述**: 给予1层虚弱。你在这个回合内每次对该名敌人造成伤害时，额外给予1层虚弱。消耗。
- **升级**: 初始虚弱层数提升至2层。

#### 混乱打击 (Chaos Strike)
- **类型**: 攻击
- **耗能**: 1
- **描述**: 造成7点伤害2次。丢弃1张牌。当你打出其他牌时，将这张牌从手牌中丢弃。
- **升级**: 伤害提升至9点2次。

#### 针锋相对 (Counterstrike)
- **类型**: 攻击
- **耗能**: 3
- **描述**: 造成25点伤害。如果敌人的意图是攻击，额外造成一次伤害。
- **升级**: 伤害提升至29点。

#### 扎透 (Puncture)
- **类型**: 攻击
- **耗能**: 2
- **描述**: 奇巧。造成2点伤害6次。通过奇巧打出时，改为造成12点伤害。
- **升级**: 伤害次数提升至8次，奇巧伤害提升至16点。

### 稀有卡牌

#### 炼金术 (Alchemy)
- **类型**: 技能
- **耗能**: 3
- **描述**: 奇巧。饥渴。获得30金币。
- **升级**: 获得的金币提升至35。
- **备注**: 饥渴 - 打出这张牌后，需要杀死一名非爪牙敌人，才能再次打出这张牌。

#### 刀山 (Blade Mountain)
- **类型**: 能力
- **耗能**: 2
- **描述**: 所有获得小刀的效果改为获得1张巨刀。
- **升级**: 获得的巨刀升级。

#### 巨刀 (Great Blade)
- **类型**: 攻击
- **耗能**: 2
- **描述**: 造成15点伤害。保留。消耗。这张牌被视为小刀。
- **升级**: 伤害提升至20点。
- **备注**: 无法通过常规方式获得，只能通过刀山能力获得。

#### 钻心 (Heart Piercer)
- **类型**: 能力
- **耗能**: 2
- **描述**: 小刀额外给予1层中毒。将4张小刀添加到你的手牌。
- **升级**: 给予的中毒层数从1层提升至2层。

#### 厄咒 (Bane)
- **类型**: 技能
- **耗能**: 1
- **描述**: 保留。获得1层无实体和9层中毒。在这个回合每受到一次攻击，减少2层中毒。消耗。
- **升级**: 耗能降为0，获得的中毒从9层减少到7层。

#### 镇静剂 (Sedative)
- **类型**: 技能
- **耗能**: 1
- **描述**: 迟钝。抽牌直到抽满手牌。这张牌在本场战斗中的耗能增加1。
- **升级**: 耗能降为0。
- **备注**: 迟钝 - 这张牌无法被丢弃。

## 新增关键字

### 饥渴 (Thirsty)
打出带有饥渴关键字的卡牌后，需要杀死一名非爪牙敌人，才能再次打出这张牌。

### 迟钝 (Sluggish)
带有迟钝关键字的卡牌无法被丢弃。

## 其他修改

### 中毒预览修复
修复了玩家拥有无实体时，中毒伤害预览显示不正确的问题。原版游戏只考虑敌人中毒的情况，未考虑玩家中毒与无实体的交互。现在预览会正确显示：
- 1层无实体：敌方回合结束后消失，显示完整中毒伤害
- 2层及以上无实体：每次中毒伤害上限为1

## 开发

### 环境要求

- Godot 4.5.1 (Mono 版本)
- .NET 9.0 SDK

### 编译

```bash
dotnet build -c Release
```

编译后会自动将 DLL 和 JSON 文件复制到游戏 mods 目录。

### 导出 PCK

编译完成后，需要导出 PCK 文件（包含图片等资源）：

```bash
# Windows
"<Godot路径>\Godot_v4.5.1-stable_mono_win64.exe" --headless --export-release "PCK" "<游戏路径>\mods\UltimateSilentCardExpansion\UltimateSilentCardExpansion.pck"
```

或者在 Godot 编辑器中：
1. 打开项目
2. 选择 **项目** → **导出**
3. 选择 "PCK" 预设
4. 点击 **导出**
